# 国创睿峰智能图书馆管理系统 - UI设计需求文档

## 一、项目概述

### 1.1 项目背景
为中心小学打造的AI智慧图书馆管理系统，服务于教师和学生两大用户群体。系统融合人工智能技术，提供智能推荐、智能检索、行为分析等创新功能，打造现代化、智能化的数字图书馆体验。

### 1.2 设计目标
- **简约现代**: 界面清爽简洁，符合现代教育场景审美
- **活力青春**: 色彩明快有活力，贴近学生群体
- **智能友好**: 突出AI功能，体现科技感和智能化
- **易用性强**: 操作简单直观，降低学习成本
- **响应式设计**: 支持PC端、平板、手机等多终端

## 二、设计风格定位

### 2.1 视觉风格关键词
- 简约 (Minimal)
- 活力 (Energetic)
- 现代 (Modern)
- 智能 (Smart)
- 友好 (Friendly)
- 教育 (Educational)

### 2.2 色彩系统

#### 主色调 (Primary Colors)
```
- 主品牌色: #667EEA (科技紫 - 体现智能化)
  用途: 主要按钮、重点信息、导航激活态

- 辅助色: #764BA2 (深紫 - 渐变搭配)
  用途: 渐变背景、hover状态、强调元素
```

#### 辅助色系 (Secondary Colors)
```
- 活力橙: #FA8C16 (代表活力和创造力)
  用途: 警告提示、重要标记、活动标签

- 成功绿: #52C41A (代表成长和正向反馈)
  用途: 成功提示、完成状态、积分增长

- 信息蓝: #1890FF (代表知识和探索)
  用途: 信息提示、链接、数据可视化

- 错误红: #F5222D (代表错误和警示)
  用途: 错误提示、删除操作、超期提醒
```

#### 中性色系 (Neutral Colors)
```
- 标题黑: #303133 (主要文本)
- 正文黑: #606266 (正文内容)
- 次要灰: #909399 (辅助信息)
- 占位灰: #C0C4CC (占位文本)
- 边框灰: #DCDFE6 (分割线、边框)
- 背景灰: #F5F7FA (卡片背景)
- 页面白: #FFFFFF (主背景)
```

#### AI功能专用色
```
- AI紫渐变: linear-gradient(135deg, #667EEA 0%, #764BA2 100%)
  用途: AI功能标识、智能推荐模块、AI助手

- AI光效: rgba(102, 126, 234, 0.1) - rgba(102, 126, 234, 0.3)
  用途: AI功能区背景、智能提示卡片
```

### 2.3 字体系统

#### 中文字体
```
- 标题/重要信息: 苹方 (PingFang SC) / 微软雅黑
  - H1: 28px, 700 (页面主标题)
  - H2: 24px, 600 (模块标题)
  - H3: 20px, 600 (卡片标题)
  - H4: 18px, 600 (小节标题)

- 正文内容: 苹方 (PingFang SC) / 微软雅黑
  - 大正文: 16px, 400 (主要内容)
  - 正文: 14px, 400 (常规文本)
  - 小字: 12px, 400 (辅助说明)
```

#### 数字/英文字体
```
- 数据展示: DIN Pro / Helvetica Neue
  用途: 统计数字、图表、数据面板

- 代码/系统: Consolas / Monaco
  用途: 系统ID、借阅证号、ISBN
```

### 2.4 圆角与阴影

#### 圆角规范
```
- 大卡片: 12px (主要内容卡片)
- 按钮/小卡片: 8px (按钮、标签)
- 输入框: 4px (表单输入)
- 圆形元素: 50% (头像、圆形按钮)
```

#### 阴影系统
```
- 悬浮卡片: 0 4px 12px rgba(0, 0, 0, 0.08)
- 弹出层: 0 8px 24px rgba(0, 0, 0, 0.12)
- 按钮hover: 0 4px 16px rgba(102, 126, 234, 0.25)
- AI模块: 0 8px 32px rgba(102, 126, 234, 0.15)
```

### 2.5 间距系统
```
- 4px   - 最小间距 (图标与文字)
- 8px   - 紧密间距 (标签间隔)
- 12px  - 小间距 (表单项间隔)
- 16px  - 常规间距 (卡片内边距)
- 20px  - 中等间距 (模块间距)
- 24px  - 大间距 (区域间距)
- 32px  - 超大间距 (页面区块)
```

## 三、核心界面设计需求

### 3.1 登录页面

#### 设计要点
- 左右分栏布局，左侧展示品牌形象，右侧登录表单
- 左侧使用渐变背景 + 插画/3D元素，体现科技感
- 支持学生/教师两种角色切换
- 提供账号密码登录、扫码登录两种方式

#### 必需元素
```

- Logo + 系统名称
- Slogan: "AI赋能阅读，智慧点亮未来"

右侧表单:
- 角色切换Tab (学生/教师)
- 用户名输入框 (学生: 学号, 教师: 工号)
- 密码输入框 (带显示/隐藏切换)
- 记住密码 checkbox
- 登录按钮 (主要按钮样式)
- 扫码登录入口 (二维码图标)
- 忘记密码链接
```

#### 交互细节

- 登录按钮hover有放大效果

## 四、组件设计规范

### 4.1 按钮组件

#### 主要按钮 (Primary)
```
样式:
- 背景: 品牌色渐变 #667EEA → #764BA2
- 文字: 白色，14px
- 圆角: 8px
- 内边距: 12px 24px
- hover: 阴影加深 + 轻微放大
- active: 略微缩小
- disabled: 灰色 + 降低透明度
```

#### 次要按钮 (Secondary)
```
样式:
- 背景: 白色
- 边框: 1px 品牌色
- 文字: 品牌色，14px
- hover: 浅色背景 + 边框加深
```

#### 危险按钮 (Danger)
```
样式:
- 背景: #F5222D
- 文字: 白色
- hover: 背景加深
```

#### 文字按钮 (Text)
```
样式:
- 无背景
- 文字: 品牌色
- hover: 下划线
```

#### AI特殊按钮
```
样式:
- 背景: AI紫渐变
- 图标: ✨/🤖
- 光效动画
- 文字: "AI识别" "智能推荐"
```

### 4.2 表单组件

#### 输入框 (Input)
```
样式:
- 边框: 1px #DCDFE6
- 圆角: 4px
- 高度: 40px
- 内边距: 0 12px
- focus: 品牌色边框 + 阴影
- error: 红色边框 + 错误提示
```

#### 下拉选择 (Select)
```
样式:
- 同输入框基础样式
- 右侧下拉图标
- 下拉面板: 白色背景 + 阴影
- 选项hover: 浅色背景
- 多选: 标签展示
```

#### 开关 (Switch)
```
样式:
- 关闭: 灰色背景
- 开启: 品牌色渐变背景
- 圆形滑块
- 动画: 0.3s ease
```

#### 日期选择器
```
样式:
- 日历面板: 白色 + 阴影
- 当前日期: 品牌色圆形背景
- 选中范围: 浅色背景
- 快捷选择: "今天" "本周" "本月"
```

#### 上传组件
```
拖拽区域:
- 虚线边框
- 中央图标 + 提示文字
- 拖入时: 高亮边框
- 上传进度条: 品牌色
```

### 4.3 反馈组件

#### 消息提示 (Message)
```
样式:
- 顶部居中弹出

- 图标 + 文字
- 成功: 绿色
- 警告: 橙色
```

#### 对话框 (Modal)
```
样式:
- 居中弹出
- 遮罩: rgba(0,0,0,0.5)
- 内容: 白色卡片
- 圆角: 12px
- 标题栏 + 内容区 + 底部按钮

```

#### 抽屉 (Drawer)
```
样式:
- 从右侧滑入
- 宽度: 600px

- 内容区可滚动
```

#### 表格 (Table)
