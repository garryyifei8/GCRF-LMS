## 1. 原型设计概述

### 1.1 原型设计原则

| 原则 | 说明 | 应用 |
|------|------|------|
| **任务导向** | 围绕用户核心任务设计页面流程 | 借还流程3步内完成 |
| **信息清晰** | 关键信息突出显示,层级分明 | 使用视觉层级和留白 |
| **操作高效** | 减少操作步骤,提供快捷方式 | 扫码输入、快捷按钮 |
| **反馈及时** | 每个操作都有明确反馈 | 成功提示、错误提示、加载状态 |
| **容错性强** | 支持撤销、二次确认 | 删除前确认、支持撤销 |

### 1.2 布局规范

**Web管理端布局**:
- 侧边栏宽度: 200px (展开) / 64px (收起)
- 顶部导航栏高度: 64px
- 主内容区最小宽度: 1024px
- 卡片间距: 16px
- 内容区边距: 24px

**小程序端布局**:
- 屏幕宽度: 375px (基准)
- 顶部导航栏高度: 44px + 状态栏
- 底部Tab栏高度: 50px
- 卡片间距: 12px
- 内容区边距: 16px

#### 2.3.2 交互说明

**步骤1: 验证读者**
- 输入读者证号 或 点击"人脸识别"
- 验证成功 → 显示读者信息 (步骤2)
- 验证失败 → 红色提示"读者证不存在"

**步骤2: 读者信息展示**
- 显示读者照片、姓名、证号、借阅规则
- 检查借阅资格:
  - ✓ 正常: 绿色标识
  - ✗ 超期: 红色提示"有图书逾期,请先归还"
  - ✗ 达上限: 橙色提示"已达借阅上限X本"

**步骤3: 扫描图书**
- 支持连续扫描
- 每扫描一本,自动添加到借阅清单
- 自动检查:
  - 图书是否存在
  - 图书是否可借 (在架状态)
  - 是否超过借阅上限

**步骤4: 确认借阅**
- 显示本次借阅清单
- 支持删除单本图书
- 点击"确认借阅" → 弹窗二次确认 → 提交
- 借阅成功 → 绿色提示"借阅成功,已借X本"
- 可选打印借阅凭证

#### 2.5.2 交互说明

**智能编目流程**:
1. 扫描/输入ISBN → 点击"查询"
2. 系统从云端检索图书数据 (显示加载中)
3. 找到数据 → 自动填充书名、作者、出版社等
4. AI自动分类 → 生成中图分类号
5. AI自动标签 → 生成关键词和主题标签
6. 用户核对信息 → 可手动修改
7. 点击"防重检查" → 检查是否已编目
8. 无重复 → 点击"保存" → 编目成功

**AI辅助功能**:
- **AI分类**: 点击"AI分类"按钮,根据书名和简介自动生成分类号
- **AI标签**: 自动提取关键词生成标签,可手动添加/删除

**批量导入**:
- 下载Excel模板
- 填写图书信息
- 上传Excel文件
- 系统解析并验证数据
- 预览导入列表
- 确认导入

- 打字动画效果 (AI回复时)
- 语音输入按钮 (长按说话)
- 发送按钮 (点击发送)
- 常见问题快捷按钮

## 5. 响应式设计

### 5.1 Web管理端响应式断点

| 断点 | 屏幕宽度 | 布局调整 |
|------|---------|---------|
| **Desktop** | ≥1280px | 三栏布局: 侧边栏 + 主内容 + 右侧栏 (如有) |
| **Laptop** | 1024px ~ 1279px | 侧边栏可收起, 主内容区占据剩余空间 |
| **Tablet** | 768px ~ 1023px | 侧边栏默认收起, 主内容区全屏 |
| **Mobile** | <768px | 不推荐移动端访问管理端 (提示使用PC) |

### 5.2 小程序响应式适配

**基准宽度**: 375px (iPhone 6/7/8)
**适配方案**: rpx单位 (750rpx = 375px)

**不同屏幕适配**:
- 375px: 基准,1:1
- 414px: 放大,内容自适应
- 320px: 缩小,保证可用性
