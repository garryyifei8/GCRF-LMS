## 🛠️ 常用命令

### Docker 操作
```bash
# 查看容器状态
docker ps | grep gcrf-library-web-admin

# 查看容器日志
docker logs gcrf-library-web-admin

# 重启容器
docker-compose restart

# 停止容器
docker-compose stop

# 删除容器
docker-compose down

# 重新构建并启动
docker-compose up -d --build
```

### 访问测试
```bash
# 测试健康检查
curl http://localhost:3000/health

# 测试首页
curl -I http://localhost:3000
```
